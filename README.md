# 📈 React Investment Calculator

A modern, interactive Investment Calculator built with React.js. This application allows users to project their investment growth over time by calculating returns based on initial investments, annual contributions, expected return rates, and duration.

## ✨ Features

- **Real-Time Calculations:** Instantly calculates and updates projected returns as you modify the input fields.
- **Data Validation:** Ensures accurate forecasting by preventing calculations on invalid durations (minimum 1 year requirement).
- **Component-Based Architecture:** Clean and maintainable codebase separated into logical React components.
- **Dynamic State Management:** Utilizes React Hooks (`useState`) for seamless data handling and UI rendering.

## 🚀 Tech Stack

- **Frontend:** React.js
- **Styling:** CSS3
- **Logic:** JavaScript (ES6+)

## 🛠️ Installation & Setup

To run this project locally on your machine, follow these instructions:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/HafizAbdulHannan/Investment-Calculator.git](https://github.com/HafizAbdulHannan/Investment-Calculator.git)


2. **Navigate to the project directory:**
   ```bash
   cd investment-calculator


3. **Install the dependencies:**
   ```bash
   npm install


4. **Start the development server:**
   ```bash
   npm run dev


5. **Project Structure:**

   src/
├── components/
│   ├── Header.jsx       # Displays the app logo and main title
│   ├── UserInput.jsx    # Handles user input fields for investment parameters
│   └── Results.jsx      # Computes and renders the calculated investment data table
├── App.jsx              # Main application container and state management logic
├── App.css              # Global and layout styling
└── main.jsx             # Application entry point
